# Michele Geronazzo

Associate Professor of Computer Engineering at the University of Padua. My research focuses on
immersive audio, virtual and augmented reality, and multimodal interaction.

- Website: https://mgero.github.io
- Email: geronazzo.michele@gmail.com

## About

This repository contains the source for my personal academic website. It is built with
[al-folio](https://github.com/alshedivat/al-folio) and Jekyll (a static site generator that converts
Markdown into a complete website).

## Research Interests

- Immersive audio and spatial sound
- Virtual and augmented reality
- Multimodal and haptic interaction
- Audio-driven perception and UX

## Content Map

- `_pages/` Static pages (Markdown, a simple text format for writing content) with frontmatter, the metadata block between `---` lines
- `_posts/` Blog posts (Markdown with frontmatter)
- `_projects/` Research projects and demos
- `_news/` News and announcements
- `_teachings/` Teaching and course entries
- `_bibliography/` Publications in BibTeX (a standard format for publication metadata)
- `_data/` Site data and CV sources in YAML (a simple configuration format)
- `assets/` Images, PDFs, audio, and other media
